# NTG Frontend Academy Materials

Official repository for NTG Academy's frontend development program. Includes course materials, live coding examples, assignments, and resources covering introduction to Web, HTML5, CSS3, JavaScript, Angular, and modern web development practices.

## 📚 Course Overview

This repository contains all teaching materials and code samples for the frontend development program at NTG Academy. Students will progress from web fundamentals to building modern web applications with Angular.

## 🚀 Getting Started

### Prerequisites

Before starting the course, make sure you have the following installed:

- **Code Editor**: [Visual Studio Code](https://code.visualstudio.com/) (recommended)
- **Web Browser**: Latest version of Chrome, Firefox, or Edge
- **Git**: [Download Git](https://git-scm.com/downloads)
- **Node.js**: Version 18+ [Download Node.js](https://nodejs.org/)
- **Angular CLI**: Install globally with `npm install -g @angular/cli`

### Cloning the Repository

```bash
git glone https://github.com/Khaterx/frontend-ntg-academy-materials.git
cd frontend-ntg-academy-materials
```

### Running Examples

Each module contains runnable examples. To run them:
**For HTML/CSS/JS examples:**
- Simply open the HTML file in your browser
- Or use Live Server extension in VS Code

**For Angular projects:**
```cmd
cd 05-angular/examples/example-name  
npm install  
ng serve
```
- Navigate to `http://localhost:4200`
